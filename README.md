# Тест швидкості
#### Даний модуль вимірює швидкість інтернету в користувацькому режимі
#### Установка:
Папку SpeedTest перемістити в /usr/abills/Abills/modules/

Папку test_data перемістити в /usr/abills/Abills/templates/

В файлі **config.pl**:
```
@MODULES = (
             'SpeedTest'
           );
```
